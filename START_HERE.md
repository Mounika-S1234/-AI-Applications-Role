## 🚀 Quick Start (5 Minutes)

### Step 1: Backend Setup
```bash
cd backend
python -m venv venv
venv\Scripts\activate              # Windows
source venv/bin/activate           # Mac/Linux
pip install -r requirements.txt
```

### Step 2: Configure API Key
```bash
# Edit backend\.env
GROQ_API_KEY=your_groq_api_key_here
DATABASE_URL=postgresql://user:password@localhost/crm_hcp_db
```

**Get Groq API Key**: https://console.groq.com/

### Step 3: Start Backend
```bash
uvicorn app.main:app --reload
```
Backend runs at: http://localhost:8000

### Step 4: Frontend Setup (New Terminal)
```bash
cd frontend
npm install
npm start
```
Frontend runs at: http://localhost:3000

## 🆘 Common Issues & Solutions

| Issue | Solution |
|-------|----------|
| "Module not found" | Run `pip install -r requirements.txt` |
| "GROQ_API_KEY not found" | Add key to backend/.env |
| "Cannot GET /" on frontend | Check REACT_APP_API_URL in .env |
| "Database connection error" | Verify DATABASE_URL, start DB |
| "Port 8000 already in use" | Change port or kill process |
| "npm install fails" | Delete node_modules, try again |
